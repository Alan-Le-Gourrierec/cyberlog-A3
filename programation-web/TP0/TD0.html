TP0 : Introduction au langage HTML

Remarque :

  1. Pour écrire votre première page web, tout ce que vous avez besoin est un navigateur pour visualiser la page et un éditeur de texte tels que notpad++,vs code ....
  2. Dans ce TP0 on va utiliser les balises basiques de HTML surtout :
    • Formulaire : Form, label, input, textarea button, select, option, fieldset et legend
    • Tableau : table, tr, td, ...
  3. La saisie des champs suivis par * est obligatoire
  4. La saisie de la ville est semi-automatique (utiliser les balises datalist et option)
  5. Lorsqu'on soumet le formulaire le contenu sera envoyé à votre adresse mail
